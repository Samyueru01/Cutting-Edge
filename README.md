# Cutting Edge Finance Project

Replication and application of the paper:

Raissi, Perdikaris, Karniadakis (2019)
"Physics-Informed Neural Networks"

## Project Goals

1. Understand PINNs methodology
2. Replicate core experiments
3. Apply PINNs to financial PDEs (Black–Scholes)

## Repository Structure

pinns_reference/ : original implementation from authors  
src/ : our implementation  
notebooks/ : exploratory experiments  
experiments/ : results and logs
